# LibroRefugio-Back
BackEnd LibroRefugio Vue

## Paso 1.-
Instalacion de la paqueteria de node 
```sh
npm install
```

## Paso 2.-
crear archivo .env definiendo el puerto con el que trabajara el servidor (basarse en .env_example)

## Paso 3.-
puesta en marcha del servidor back
```sh
npm run serve
```
